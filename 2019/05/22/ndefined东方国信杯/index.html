###东方国信杯

Heading

直接贴题：

1、针对马拉松数据（marathon-data.csv），进行数据分析，统计不同年龄性别的人马拉松成绩并用图表展示。
 题目：比较不同性别、年龄选手的成绩差异用图表展示，并提供你对数据的解读。

由于是小组作业，选择了一个自己稍微擅长的。这个一看就是画画图，随便写两句就行的题目。况且我还有Matlab，直接读取，画图。听起来分分钟搞定的事情，我居然做了3个小时。

主要是Matlab不熟练，一些基本的语法代码都忘了。昨天被某位小可爱实名diss说我记性不好（手动狗头）我还是老老实实在这里重新记录一下：

最简单的，从excel中读取数据：
h=xlsread(‘工作簿2.xlsx’,’sheet5’,’A2:A12586’);
写一个全一的列向量：
A=ones(80,1); 区分行列可以这么记 ，先行后列，80为行，1为列。需要的话，转置一下。
写一个全零的列向量：
A=zeros(80,1)；
Matlab数组的取法：
是A(1)不是A[1]
还有加标签：
title(‘女性61-83岁split’);
ylabel(‘分钟’);
多图画法：
subplot(4,2,1)
还有我还会excel中的

做到这发现画出来的图不具有特点，发现不了规律。差一个每个年龄跑马拉松时间数的表，要是数据少就用excel做了，不对，数据少也不要，现在巨讨厌重复性的工作。
好长没写代码，手都生了：
简单的都忘了，循环忘了，每次的sum没重新初始化。。。。
（多刷题吧，卑微。。。）

function excel(~)
fn = ‘工作簿2.xlsx.sheet5’;

b=xlsread(‘工作簿2.xlsx’,’sheet5’,’F2:F12586’);

Mb=xlsread(‘工作簿1.xlsx’,’sheet5’,’F2:F24666’);
A=ones(80,1);
